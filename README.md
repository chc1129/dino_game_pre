# dino_game_pre

Chrome Dinosaur Game をjavascriptの勉強とAIの利用法の勉強を兼ねて  
dino game preを作成

遊び方  
ファイル一式をダウンロード  
任意のフォルダに格納されたindex.htmlをブラウザで開く  
ゲーム開始はスペースボタン押下  
ゲーム再開始は、F5押下かページ更新し、スペースボタン押下
